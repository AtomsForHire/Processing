# Processing

Python program to get various metrics of hyperdrive calibration solutions and images.

Requires: https://github.com/Chuneeta/mwa_qa

requires data.in file formatted like this:

working dir
save rms dir
save cal dir
save smoothness dir
dir to calibration metafits

Example:

'./'
'./rms_cal/both/'
'./var_cal/both/'
"./smoothness/"
"../rerun_1_solutions/"
